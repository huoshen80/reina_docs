# 添加游戏

支持手动选择游戏可执行文件和拖拽文件夹或可执行文件两种方式添加游戏

## 手动选择可执行文件

1. 打开游戏添加界面后，点击选择启动程序，选择游戏本体的可执行文件

![选择可执行文件](https://img0.huoshen80.top/i/2026/01/02/6957abeda5a8d.png)

2. 选择用于刮削数据的数据源，推荐使用 Kun 或 Mixed

3. 检查自动获取到的游戏名称是否正确，确认无误后点击确认

4. 查看刮削结果是否正确，如果部分数据源刮削有误，可以进入[游戏详情页编辑](editgame.md)修改数据源的相关部分

*确认弹窗*
![确认弹窗](/images/addgame_20260619_214122.png)

*选择弹窗*
![选择弹窗](/images/addgame_20260619_214314.png)


## 拖拽文件夹或者可执行文件

直接将游戏文件夹或者游戏可执行文件拖拽到 Reina 内，松开鼠标后会自动进入添加流程

## 批量导入

1. 在游戏添加界面点击批量导入，选择一个包含所有游戏文件夹的目录，系统会自动扫描该文件夹下的所有的游戏并列出
2. 在弹窗右上角选择一个数据源（可选），设置扫描深度（可选），然后点击`匹配元数据`
3. 等待匹配完成后，可以手动修改匹配错误或者未找到的条目，也可以修改游戏的启动程序
4. 最后点击`导入已匹配`按钮，完成批量导入
![批量导入](/images/addgame_20260619_214545.png)

PS：`待处理`、`未找到`、`错误`等状态的游戏可以被导入为自定义游戏，自定义游戏只包含游戏名称和启动程序，其他信息需要前往[游戏详情页编辑](editgame.md)手动填写（也可以通过 id 刮削）

## 添加模式

Reina 添加游戏有多种模式，默认使用游戏名称搜索，此外还支持 ID 搜索，自定义游戏

#### 自定义模式

想导入数据源可能没有的游戏时，可以选择自定义模式，需要前往[游戏详情页编辑](editgame.md)手动填写游戏的所有信息

#### ID 搜索

你可以在输入框中填入任意一个数据源的游戏 ID

如何获取 bgm、vndb、ymgal 和 kun 上的游戏 ID：
例如打开[Vndb](https://vndb.org/)，搜索对应游戏，进入游戏页面后，在网址栏中可以看到对应的 ID
