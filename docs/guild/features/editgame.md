# 编辑游戏资料

如果你发现游戏资料有误，或者想要自定义游戏的封面、名称、Tag、开发商等元数据，可以通过编辑游戏资料来实现，在这里你还可以使用游戏 ID 重新刮削游戏资料

### 切换显示源
修改数据源后点击`切换显示源`按钮

### 名称搜索

此功能同单个添加游戏的流程，搜索框可以用回车触发搜索（单源也可以直接输入 ID 来搜索）

### 手动 ID

有不想要的数据源，可以删除其对应的 ID，然后点击`从数据源更新数据`按钮

### 使用单一 ID 重新进行 Mixed 刮削

数据源为 Mixed 的情况下，如果在手动 ID 模式下只填一个数据源的 ID，点击从数据源更新数据按钮后，Reina 将会自动填充剩余数据源的 ID

## 游戏资料编辑

在游戏资料编辑界面，你可以选择或自定义游戏的名称、封面、开发商、Tag 等元数据，修改后记得划到页面底部点击`保存所有更改`按钮
