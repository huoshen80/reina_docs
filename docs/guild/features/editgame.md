# 编辑游戏资料

如果你发现游戏资料有误，或者想要自定义游戏的封面、名称、Tag、开发商等元数据，可以通过编辑游戏资料来实现，在这里你还可以使用游戏 ID 重新刮削游戏资料

## 数据源更新

![数据源更新](/images/editgame_20260222_172944.png)

#### 剔除错误数据源

如果你发现哪个数据源刮削到的资料有误，你可以删除对应的游戏 ID，然后点击从数据源更新数据按钮

#### 使用单一 ID 重新进行 Mixed 刮削

数据源为 Mixed 的情况下，如果只填写一个数据源的 ID，点击从数据源更新数据按钮后，Reina 将会自动填充剩余数据源的 ID

## 游戏资料编辑

在游戏资料编辑界面，你可以修改或自定义游戏的名称、封面、开发商、Tag 等元数据，修改后记得划到页面底部点击保存所有更改按钮
