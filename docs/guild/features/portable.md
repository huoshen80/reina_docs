# 便携模式

## 便携模式的启用条件

当 reina 可执行文件所在目录下存在 `resources/data` 文件夹时，reina 将启用便携模式。

## 便携模式的特点

如果在没有设置存档备份路径、数据库备份路径的情况下，便携模式默认文件存放位置如下：

``` shell
resources/
├── backups/            # 游戏存档备份文件夹
├── covers/             # 游戏封面文件夹
└── data/               # 数据库文件夹
    └── backups/        # 数据库备份文件夹  
```

## 从安装版切换到便携版

如果想要切换到便携模式，可以按照以下步骤操作：

移动 `%AppData%\com.reinamanager.dev` 文件夹下除 `.window-state.json` 之外的文件到 `resources` 文件夹下，`resources` 一般需要手动创建。

## 关于便携版的更新

- 从一开始就在 Github 上下载的便携版，需要更新时直接下载新版本的便携包覆盖原有文件即可。

- 从安装版切换到便携版的用户，如果没有移动 reina 的位置，可以直接点击立即更新按钮进行更新。
