# 一键转区与自动放大

## 启用方式

在游戏详情页右上角**更多**按钮中
![转区与放大](/images/le-magpie_20260223_120823.png)

如果没有提前设置转区软件和 Magpie 软件路径，会自动弹出路径设置弹窗

对于 LE 来说选择 `LEProc.exe`，对于 Magpie 来说选择 `Magpie.exe`

## Magpie 注意事项

由于 Magpie 的自动放大功能是基于最新版本默认快捷键（Shift + Win + A）来实现的，所以请确保你的快捷键没被修改过

此外对于启动过慢的游戏，自动放大功能可能会失效
