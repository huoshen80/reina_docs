# 功能概述

## 游戏管理

- **多种添加游戏方法：** 支持[拖拽添加](addgame.md)、[WhiteCloud 导入](whitecloud.md)
- **自定义元数据：** 允许用户自定义游戏封面、名称、Tag、开发商等
- **游戏状态管理：** 支持标记游戏为想玩、在玩、玩过、搁置、弃坑
- **搜索：** 支持通过游戏名称、别名、开发商等元数据进行搜索
- **排序与筛选：** 支持最近游玩、最近添加、评分等多种排序方式，支持本地游戏、网络游戏等多种筛选方式
- **游戏时长追踪：** 支持真实游戏时间（默认）和游戏启动时间两种模式
- **游戏存档备份：** 支持自动或手动备份游戏存档，提供一键恢复功能
- **一键转区和超分放大：** 游戏详情页右上角更多按钮内部提供对应功能的开关
- **NSFW 过滤：** 通过设置内简单的开关隐藏或遮盖 NSFW 内容
- **批量操作：** 支持从 API 批量导入、添加和更新游戏元数据

## 收藏夹

- **分组与分类：** 支持创建分组，在分组内创建任意分类，进入分类或者右键分类开始管理你的收藏夹
- **开发商分类：** 在收藏夹页面开发商内自动根据刮削数据对游戏的开发商进行分类
- **游戏状态分类：** 在收藏夹页面游戏状态内自动根据游戏状态进行分类
- **拖拽排序：** 支持在自建分类内通过拖拽来调整游戏卡片的顺序

## 游玩记录统计
- **详细的游玩时间统计：** 提供每个游戏的总游玩时间、游玩次数、今日游玩时间等
- **可视化图表：** 提供直观的折线统计图展示游玩时间的分布和趋势、支持自定义的时间范围
